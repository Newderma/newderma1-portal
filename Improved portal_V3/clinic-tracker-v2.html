Clinic Tracker Setup
One-time configuration

Connect your free Supabase database so all staff share data in real time from any device.

1
Create a free Supabase account Go to supabase.com → New Project → choose a name and region closest to your clinic.
2
Run this SQL to create the database tables In Supabase → SQL Editor → paste this → Run:
-- Clinic Tracker Database Setup CREATE TABLE ct_config ( key text PRIMARY KEY, value jsonb ); CREATE TABLE ct_records ( id bigint PRIMARY KEY, date text, patient text, patient_id text, doctor text, procedure text, nurse text, notes text, items jsonb, total_sell numeric, total_cost numeric, profit numeric, margin numeric, created_at timestamptz DEFAULT now() ); CREATE TABLE ct_vacation ( id bigint PRIMARY KEY, employee text, from_date text, to_date text, days int, type text, note text, req_date text, status text DEFAULT 'Pending', reject_reason text, decided_at text ); -- Allow public access (the app handles auth itself) ALTER TABLE ct_config ENABLE ROW LEVEL SECURITY; ALTER TABLE ct_records ENABLE ROW LEVEL SECURITY; ALTER TABLE ct_vacation ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON ct_config FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all" ON ct_records FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all" ON ct_vacation FOR ALL USING (true) WITH CHECK (true);
3
Get your Project URL and anon key In Supabase → Project Settings → API → copy Project URL and anon public key.

Your key is stored only in this browser. Supabase free tier supports up to 50,000 rows — more than enough for a clinic.
